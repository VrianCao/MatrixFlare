# EVID-GOV-001 Artifact Contract

本目录承载治理门禁 `EVID-GOV-001` 的 CI 产物。

每次运行应在 `<run_ts>/` 下生成至少以下文件：

* `summary.md`
* `artifacts/requirement-register.csv`
* `artifacts/requirement-register.json`
* `artifacts/traceability-matrix.csv`
* `artifacts/traceability-matrix.json`
* wildcard route family 的实际展开审计快照

规范语义以以下分册为准：

* `spec/framework/14-traceability-and-change-control.md`
* `spec/framework/43-testing-and-compliance.md`
* `spec/framework/44-verification-and-evidence-register.md`
