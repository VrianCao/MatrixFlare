## Decision

当前项目正式接受 same-head GitHub Actions run `23986289866`（head `b6b3789eebf51f853cc5b96f295890e644cef553`）的 raw `pre-release` artifact，作为关闭 [OQ-0005](/root/Matrix/spec/open-questions/OQ-0005.md) 的充分依据。

该接纳只回答 `OQ-0005` 的问题本身：current pushed head 是否已能在真实 Cloudflare gradual deployment + named-environment topology 下，产出 truthful rollout-skew proof。被接受的最小事实基线如下：

* `local-foundation` 成功，workflow 真实进入 same-head non-local path；
* `nonlocal (pre-release)` 成功完成 `Prepare non-local ops Access session`、dual-version rollout start/restore、suite execution、raw bundle upload、R2 upload 与 provenance write；
* raw `pre-release.log` 中，`TEST-OPS-001` 为 `pass`；
* raw artifact 中已真实产出 `rollout_skew_probe.json`；
* `RolloutSkewProbeResponse.assertions.new_worker_old_authority == true` 且 `old_worker_new_authority == true`；
* 同一 raw proof 已记录 baseline/candidate gateway official version IDs / tags 与 dual-version deployment ID。

本决策同时固定以下边界：

* 该 closure basis 是 same-head raw rollout-skew proof，不是 `pre-release` attestation，也不是 `EVID-OPS-001` pass；
* `EVID-OPS-001` 仍必须继续遵守 [44-verification-and-evidence-register.md](/root/Matrix/spec/framework/44-verification-and-evidence-register.md) 的 attestation-only fail-closed gate；
* run `23986289866` 已证明 `EVID-OPS-001` 当前继续失败的原因不再是 rollout-skew proof 缺失，而是 shared `pre-release` attestation 仍被 [OQ-0006](/root/Matrix/spec/open-questions/OQ-0006.md) 下的 `TEST-COST-001` / `pre_release_cost_observation` blocker 阻断。

## Rationale

* `OQ-0005` 在相关 `IF/DATA/FLOW/TEST` 资产与 dedicated canonical suite 落地之后，剩余问题已不是 product boundary 或 contract gap，而是 empirical current-head viability。
* same-head run `23986289866` 已直接回答这个 empirical question；继续把 `OQ-0005` 保持为 open 会夸大一个已经被 current-head raw artifact 收口的 blocker。
* 同时，仓库的 governance 规则要求 closed `OQ` 必须收敛到 canonical `DEC-ID`；因此必须把这条“raw proof 足以关闭 OQ、但不足以让 evidence pass”的边界显式冻结为决策，而不是只留在运行记录里。
* 通过该决策关闭 `OQ-0005`，不会放松 attestation-only fail-closed 门禁，反而能把 remaining blocker 更精确地收敛到 cost evidence。

## Consequences

* [OQ-0005](/root/Matrix/spec/open-questions/OQ-0005.md) 关闭；
* `TODO.md`、`OQ-0004`、`DEC-0003` 与 [15-source-observation-register.md](/root/Matrix/spec/framework/15-source-observation-register.md) 不得再把 rollout-skew viability 写成 active blocker；
* `TEST-OPS-001` 现可被诚实描述为“same-head raw pre-release proof 已闭环”；
* `EVID-OPS-001` 仍保持 `fail`，直到 same-head `pre-release` attestation 存在；
* `08.04` 与 `08.05` 仍未完成，因为 [OQ-0006](/root/Matrix/spec/open-questions/OQ-0006.md) 与 [OQ-0002](/root/Matrix/spec/open-questions/OQ-0002.md) 的 cost blocker 仍然存在。
